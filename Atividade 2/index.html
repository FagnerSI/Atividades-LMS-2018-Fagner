Veja como instalar e testar a beta do Windows 10 Lean, o “Windows magrinho”

Introdução

A Microsoft vem trabalhando em uma versão “mais magra” do Windows, o chamado Windows 10 Lean, criado para caber em dispositivos de entrada de 16 GB de armazenamento interno, a exemplo de laptops e tablets. A versão alternativa do sistema operacional foi distribuída junto com uma prévia do Build Redstone 5, o próximo update do Windows 10. Confira abaixo como fazer o download e ver como ele funciona.

Instalação

Antes de mais nada, é preciso lembrar que a edição atual está cheia de bugs e, claro, pode causar problemas. Portanto, estejam certos de tomarem as devidas precauções para restaurar sua máquina caso necessário, certo? Vamos lá:

  • O arquivo está hospedado no Windows Blog Italia. Você precisa fazer um registro, que é meio chatinho porque está em italiano, mas é só traduzir com o Chrome e preencher com email e nome de usuário.

  • Depois disso, entre neste link para a versão 64-bits ou neste link para a 32-bits

  • Com o arquivo você poderá criar uma versão .ISO para queimar um CD ou usar em um drive virtual. Depois disso, é só executar o arquivo de instalação como qualquer outra versão do Windows e aguardar seu término

Uma pequena tour sobre o que o Windows 10 Lean oferece

A novidade na verdade ainda não tem um nome certo e foi apenas apelidado de “lean” porque a ideia é ser mesmo uma versão mais “magra” em relação à original. Contudo, no sistema ela é identificada como 10 S Insider Preview e CloudE. O espaço total ocupado é de 6,50 GB, dos quais 6,27 GB ficam para as operações do sistema e 175 MB são dedicados a aplicativos e jogos.

Para conseguir esse corpo “fininho”, a Microsoft precisou limar drivers, apps, programas e alguns recursos. Foram mais de 50 mil arquivos deixados de lado, em relação à edição Pro. Quem quiser dar uma olhadinha na lista, acesse este endereço. O desempenho foi otimizado com a retirada do plano de fundo da área de trabalho e a presença de ferramentas embutidas no próprio Windows 10 Lean. Isso permite performance de 64-bits ocupando somente 1,5 GB de RAM e muita agilidade, mesmo em máquinas virtuais.



Galaxy S4 ganha o Android 6.0.1 Marshmallow graças a ROM personalizada”

Introdução

Samsung não deve comtemplar o Galaxy S4 com uma atualização oficial do Android Marshmallow, como vimos no vazamento de uma lista com os aparelhos elegíveis pela fabricante coreana. Mas depois de uma ROM imatura disponibilizada no fórum XDA no final de outubro, eis que agora já temos uma opção estável e com todas as características da CyanogenMod 13.

Entre as principais características da ROM personalizada podemos citar tema Dark Material, botões, mods para bateria, atalhos para a tela de bloqueio, customização da barra de status, animações, desbloqueio rápido, otimização da autonomia, e muito mais. Abaixo você confere links para download e o passo a passo para instalação.

Pré-requisitos

  • Backup de todos os dados importantes (uma limpeza completa será necessária).
  • Ativar Depuração USB em Configurações > Opções de Desenvolvimento > Depuração USB.

  • Galaxy S4 I9505 deve estar com acesso root, desbloqueado e com uma custom recovery (TWRP ou CWM).

  • O usuário deve assumir todos os riscos envolvidos no processo de atualização.

Como atualizar o Galaxy S4 i9505:

  • Arquivos necessários: baixar a ROM Ressurrection Remix com Android 6.0.1 Marshmallow para o computador (sem extrair o conteúdo).

  • Baixar também o pacote Google Apps (gapps).

  • Conectar o Galaxy S4 no computador via USB e copiar o arquivo .zip para o cartão SD do smartphone.

  • Desconecte o cabo USB e desligue o Galaxy S4.

  • Religar o smartphone em modo Recovery (pressionando e segurando simultaneamente os botões Volume para cima + Home + Power).

  • Recomenda-se realizar um backup do SO atual via opção na própria custom recovery.

  • Limpe os dados da memória interna selecionando a opção wipe data / factory reset e confirme o processo.

  • Selecione a opção install zip from sdcard, busque o arquivo da ROM Ressurrection Remix no cartão SD e confirme o processo de flash.

  • Retorne ao menu principal da custom recovery e selecione a opção para reiniciar o Galaxy S4 I9505, agora já com a nova ROM personalizada.


SVG in HTML

SVG Code explanation

  • An SVG image begins with an <svg> element
  • The width and height attributes of the <svg> element define the width and height of the SVG image
  • The <circle> element is used to draw a circle
  • The cx and cy attributes define the x and y coordinates of the center of the circle. If cx and cy are not set, the circle's center is set to (0, 0)
  • The r attribute defines the radius of the circle
  • The stroke and stroke-width attributes control how the outline of a shape appears. We set the outline of the circle to a 4px green "border"
  • The fill attribute refers to the color inside the circle. We set the fill color to yellow
  • The closing <circle> tag closes the SVG image

    • Note: Since SVG is written in XML, all elements must be properly closed!
